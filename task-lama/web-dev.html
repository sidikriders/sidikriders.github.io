"Perencanaan Sederhana Saya tentang Websites"

  1. Website yang ingin Saya Buat.

    Sejatinya website yang ingin saya buat ada berbagai banyak macam, namun untuk website official pertama saya, saya ingin membuat website tentang pariwisata dan paket wisata. sedangkan website yang benar-benar ingin saya buat adalah website mini games seperti slither.io, diep.io, agar.io, dan situs mini game sejenisnya.

  2. Hasil dan Informasi yang Ingin Saya Sampaikan

    Untuk situs pertama saya, tentu saja informasi yang ingin di sampaikan bersangkutan dengan objek wisata yang dibahas di berbagai daerah, seperti lokasi, How to Get There, budaya masyarakat sekitar, tradisi unik, hingga makanan khas daerah tersebut.

    Namun untuk situs mini games yang ingin saya buat, saya ingin menyajikan hasil yang berupa games yang bisa dimainkan bersama-sama (multiplayer), ringan, dan menagihkan

  3. Perkiraan Saya tentang Proses Pembuatan Website dari Awal Hingga Akhir.
